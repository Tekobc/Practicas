Sobre la carrera

La Tecnicatura Superior en Desarrollo de Software forma profesionales capacitados en programación, desarrollo de aplicaciones web y móviles, bases de datos, redes y gestión de proyectos tecnológicos.

Plan de Estudios

Primer Año

  • Análisis Matemático
  • Administración y Sistemas de Base de Datos
  • Introducción a la Programación
  • Introducción a las Redes
  • Sistemas Operativos
  • Laboratorio de Hardware
  • Prácticas Profesionalizantes I

Segundo Año

  • Inglés I
  • Desarrollo de Aplicativos Móviles
  • Álgebra y Lógica
  • Programación I
  • Estadística y Probabilidades
  • Desarrollo de Sistemas Orientados a Objetos
  • Prácticas Profesionalizantes II

Tercer Año

  • Inglés II
  • Gestión de Proyectos
  • Desarrollo de Sistemas Web
  • Ingeniería de Software
  • Metodología para pruebas de sistemas
  • Prácticas Profesionalizantes III

Campo Laboral

  • Desarrollo de aplicaciones web y móviles
  • Programación y desarrollo de software
  • Administración de bases de datos
  • Testing y control de calidad de software
  • Gestión de proyectos tecnológicos
